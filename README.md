# Data 8. Foundations of data science

Hi this is Kelly & this repository contains my notes, homework, and lab work as I go through the UC Berkeley Data 8 course.

### What's UCB Data 8?
Data 8 is basically an intro to data science using Python. You learn how to work with data, make graphs, do probability, testing and basic machine learning.
